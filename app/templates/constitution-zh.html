Linux 用户协会轻量级网络加速服务使用章程

version: 1.0.5

总则

USTC LUG 网络服务总则

介绍

中国科学技术大学 Linux 用户协会（以下称 LUG）所提供的轻量级网络加速服务是一项主要为科大学生与教职工提供的非营利性的网络加速服务。

功能

  • 国际网络加速

支持协议

  • HTTP 1.1
  • HTTP 2.0 TLS

申请服务

申请对象

您必须符合如下描述之一才能成为该服务的申请对象：

  • 您是科大在读本科生、研究生或博士生。
  • 您是科大在职教工或博士后。
  • 您在 LUG 中担任过任意职务。

如您满足上述描述之一，您可以在这里注册您的轻量级网络加速服务账号。

申请资格

如果您成为了申请对象，您需要满足以下条件之一来获取使用该服务的资格：

  • 您在本年度内参加了任意一次小聚或其他 LUG 举办的活动。(*)
  • 您主讲过一次小聚或 Linux User Party。(*)
  • 您曾撰写新闻稿或向 LUG Planet 投稿，并被审核通过或采纳。(*)
  • 您参与过其他在 LUG 社团活动或事务。(*)
  • 您是科大在职教工或博士后。
  • 您在 LUG 中担任过任意职务。

对于星号标记的条件，您必须声明详情，以供 LUG 核实。

申请材料

  • 姓名
  • 学号（10 位）或工号（5 位）
  • 科大邮箱（没有科大邮箱？点我注册）
  • 手机号（仅用于紧急联系）
  • 申请资格

申请资格填写说明

您只需在申请理由中陈述您满足的条件即可，无需说明用途。LUG 将根据您的申请材料做出判断，或要求您补充信息。

如果您填写的条件要求您声明详情，请务必指出。

下面列举了一些常见的不被接受的申请理由：

  • 我需要访问某些网站。
  • 科研需要。
  • 我不在校内但需要访问校内资源。

申请方式

使用校园网访问在线申请网站提交您的申请材料。LUG 将尽快予以答复。

提示：提交虚假的申请材料，将永久失去申请资格，已开通的账号亦将被封停。

有效时间

在秋季学期申请：自申请通过至下一个秋季学期开始（次年 9 月）。

在春季学期申请：自申请通过至下下一个秋季学期开始（次年 9 月）。

限制

资源限制

  • 带宽：未限制
  • 月流量：20 GB

功能限制

  • 仅允许访问 TCP 80, 443, 5228 端口

滥用

为了保障服务器安全稳定运行，运行本服务时禁止以下行为：

  • 分享账号
  • 将连接轻量级网络加速服务的设备作为网关或代理服务器，为其他主机提供网络服务（个人使用的除外）
  • 网络扫描、网络攻击或其他黑客行为
  • 运行网络爬虫
  • 可能导致 IDC 投诉的行为（如下载盗版资源）
  • 违反中国法律法规或科大校纪校规的行为
  • 管理员认定的其他滥用行为

一经发现，管理员将视情节发出警告、暂停账号或永久封停账号。

隐私

轻量级网络加速服务不提供隐私保护功能，所有经过服务器请求将被自动记录并分析，LUG 承诺不会对外泄露这些数据。记录保存时间为 180 天。

例外：当接到上级主管部门要求时，我们有权在不告知您的情况下提交这些数据。
